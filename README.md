# 🚌 BusTracker — Real-time Bus Tracking App

> Built for **Smart India Hackathon (SIH) 2025**

BusTracker is a smart, animated bus tracking companion that gives commuters real-time bus location updates, route visualization, and a seamless mobile-first experience.

---

## ✨ Features

- 🗺️ **Real-time route visualization** with animated route lines
- 🔐 **Authentication flow** — Sign In / Sign Up with smooth transitions
- 🚌 **Live bus status** with animated bus icon and stop indicators
- 📍 **Multi-stop route tracking** with ETA display
- 📱 **Fully responsive** — optimized for mobile-first use
- ⚡ **Smooth page transitions** with CSS animations (fade-in, slide-up)

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| Tailwind CSS | Styling & responsive design |
| JavaScript (Vanilla) | Interactivity & page transitions |
| SVG Animations | Bus icon & route line animations |
